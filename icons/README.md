
# Icon Files

This directory contains the icon files for the CyberCop WhatsApp Scam Detector extension.

## Required Icons:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)  
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Design Guidelines:
- Use a shield or security theme
- Include police/security colors (blue, green, red)
- Keep it simple and recognizable at small sizes
- Use transparent background for better integration
